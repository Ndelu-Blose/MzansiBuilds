# Product Positioning

MzansiBuilds helps builders discover relevant projects, evaluate momentum, prove collaboration credibility, and act on the next best step.

## Core Value
- Discover active opportunities through open roles and matching.
- Evaluate project quality through health, timeline, and trust signals.
- Prove contribution through collaboration receipts and builder reputation.
- Return with purpose through digest, notifications, and momentum surfaces.

## Why Not Just GitHub
- GitHub organizes repositories.
- MzansiBuilds organizes collaboration opportunities and credibility outcomes.

## Primary Audience
- Builders seeking projects to join.
- Owners seeking credible collaborators.
- Recruiters/employers evaluating real execution signals.
